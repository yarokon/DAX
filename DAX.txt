Обчислити лінії фігури (дві точки та напрями) {
  Точки {
    Створити нову точку (l[i-1].vn + l[i].vn) * масштаб;
    З'єднати точку з початком відповідної ліній (намалювати)
    Додати точку в масив точок;
  }

  Напрями {
    З'єднати почергово точки лініями, інфу напрямків взяти з батьківської лінії;
    Якщо відстань між точками === 0, пропустити;
    Додати лінії в масив ліній;
    Якщо ліній немає - вийти з функції;
  }

  Перетин {
    Обчислити перетин ліній;
    Утвориться від 1 до 3 нових ліній;
    Перетин двох - намалювати і викинути з масиву;
    Виправити старі лінії;
  }

  Розбиття на окремі фігури {
    Поки в масиві є лінії:
    З першої лінії шукати лінії які виходить з її кінця.
    Якщо знайшлася одна лінія - це наступна лінія в масиві;
    Якщо дві то друга;
    Додати лінію в новий масив ліній
    Зустрівши першу лінію почати наступний обхід
  }
  
  Перевірка {
    Для кожного масиву ліній перевірити послідовні лінії
    та з'єднати їх якщо вони є;
  }

  Поки в масиві є масив ліній передати їх у рекурсивну функцію
}
